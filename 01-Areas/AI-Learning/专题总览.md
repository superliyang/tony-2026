## 建议学习路径

1. 如果你在梳理整个 AI 体系，先读 [[../AI 总控制塔|AI 总控制塔]]，再读 [[07-Maps/AI 五大专题审察与组织|AI 五大专题审察与组织]]，再配合 `Canvas` 和 `Base` 看结构
2. 先在 [[../AI-Foundations/专题总览|AI-Foundations]] 建立历史、范式和基础概念
3. 再到 [[06-Topics/AI 主题索引|AI 主题索引]] 理解现代 AI 主线
4. 然后阅读 [[03-Models/模型索引|模型索引]]，把技术概念映射到代表模型
5. 如果你想先把截至 `2026-04-07` 的模型格局刷到最新，插入 [[../截至 2026-04-07 的 2026 新模型刷新|截至 2026-04-07 的 2026 新模型刷新]]
6. 如果你想先知道什么最值得重投入，把这条优先级判断线插进来：[[../最近半年最值得重投入学习的 AI 主线（截至 2026-04-07）|最近半年最值得重投入学习的 AI 主线（截至 2026-04-07）]]
7. 如果你想把最近五年的关键论文直接接回模型、系统和产品路线，先读 [[../近五年关键 AI 论文与路线映射（2021-2025）|近五年关键 AI 论文与路线映射（2021-2025）]]，再回 [[04-Papers/论文索引|论文索引]]
8. 再阅读 [[09-Systems/系统索引|系统索引]]，理解产品、平台和 runtime 这一层
9. 如果你想建立更像专家的回顾方式，插入一条 `人物 -> 公司 -> 系统 -> 案例` 的证据线：先看 [[../哪些 AI 人物最值得长期跟踪|哪些 AI 人物最值得长期跟踪]]、[[../怎么读 AI 组织：能力画像、生态位与 Adoptability|怎么读 AI 组织：能力画像、生态位与 Adoptability]]，再回 [[02-People/人物索引|人物索引]]、[[01-Companies/公司索引|公司索引]] 和 [[../AI-Applications/04-Case-Studies/案例索引|案例索引]]
10. 如果你想系统理解 Agent，先顺着 `Agent -> Prompt/Context -> Tool Use -> MCP / Browser -> Memory / Planning / Multi-Agent -> Agent 平台` 读
11. 如果你想理解 AI infra / serving，顺着 `AI 基础设施与 GPU Cloud -> Inference Serving -> AI Infra 与推理服务生态图` 读
12. 如果你想理解实验、评测、registry 和反馈闭环，顺着 `MLOps 与 LLMOps -> MLOps 与 LLMOps 生态图 -> MLflow / W&B / Langfuse / Phoenix / Promptfoo` 读
13. 如果你想走一条 `MacBook Pro Max M4` 本地专家路径，先读 `Apple Silicon 本地 AI 开发 -> MLX / Ollama / llama.cpp -> Mac AI Expert Path -> MacBook Pro Max M4 学 AI 与大模型：完整路线`
14. 如果你想理解 AI security，顺着 `AI Safety -> AI Security -> AI Safety 与 AI Security -> Prompt Injection 与 Jailbreaks -> Azure AI Content Safety / NeMo Guardrails -> AI Security Threat Map -> AI Security 控制点图` 读
15. 然后进入 [[../AI-Engineering/专题总览|AI-Engineering]] 看工程实现与治理
16. 最后进入 [[../AI-Applications/专题总览|AI-Applications]] 看业务 workflow 与组织落地
