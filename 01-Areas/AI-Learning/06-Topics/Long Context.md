# Long Context

## 这个主题是什么

`Long Context` 关注模型在超长输入、长文档处理和复杂上下文保持方面的能力。

## 为什么重要

- 真实工作很少只涉及几句短文本，往往要处理长文档、代码库、聊天历史和多文件上下文
- 长上下文能力会直接影响 assistant、coding 和 enterprise 场景

## 你先要抓住什么

- 上下文窗口更长，不代表模型就一定“真正理解得更好”
- 长上下文能力既是架构问题，也是训练与推理成本问题
- 很多任务里，检索和上下文组织方式比单纯扩大窗口更关键

## 关键问题

- 为什么上下文长度增加后，模型表现不一定线性变好
- 长 context 和 [[RAG]] 是替代关系还是互补关系
- 在 coding、文档分析、assistant 场景中，长上下文到底解决了哪些真实问题
