# OpenAI API

## 简介

`OpenAI API` 是 OpenAI 把模型能力提供给开发者和企业的主要平台接口。

## 为什么重要

- 它决定了 OpenAI 如何进入开发者生态
- 它是大量 AI 产品、agent 和自动化工作流的基础设施
- 它让模型能力从单一产品走向平台化分发

## 你可以把它当成什么来理解

- OpenAI API 是把模型能力“操作系统化”的重要一步
- 它让模型不再只是聊天界面中的能力，而是任何软件都能调用的通用能力层

## 学习时重点看什么

- 它与 [[GPT 系列]] 的关系
- API 能力如何塑造产品生态
- 它如何影响开发者对 prompt、tool use、workflow 的设计方式

## 这个平台为什么关键

- 平台决定生态，生态反过来决定模型影响力
- 很多 AI 创业和内部工具，并不是直接从模型开始，而是从 API 能力约束开始设计
- 理解 API，比单纯理解模型更能帮助你看懂“AI 如何进入软件行业”
