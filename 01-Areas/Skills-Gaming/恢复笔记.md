# 恢复笔记

## Last Session

- We upgraded the area from a strong hackathon-oriented learning system into a much more expert-oriented path by adding `UGS / PlayFab / Photon`, `game feel / onboarding / session pacing`, dispute and anti-cheat postmortem patterns, and a `90-day` expert path.

## What I Understand Now

- 你已经不只是知道谁在做 `skills gaming`，而是开始能判断：什么样的玩法更像 skill game，什么样的系统边界更可信，什么样的技术适合 V1。
- 真正让一款技能游戏成立的，不只是 tournament 和 score pipeline，还包括 `game feel`、result explainability、support/dispute path。
- 行业内前沿技术不只是 `AI NPC`；`UGS / PlayFab / Photon` 这类 liveops、experimentation、realtime infrastructure 也很关键。
- 现在这套知识库已经很接近“专家成长路径”，剩下最值钱的是做出 build、跑测试、做 postmortem。

## What Still Feels Fuzzy

- 真正的 feel 仍然要靠 build 和试玩来验证。
- `Word Sprint Duel` 的具体词库、UI hierarchy、错误惩罚和结果页节奏仍是 execution-level decisions。
- 真正的玩家争议和 support 负担，还没有经过实测数据验证。
