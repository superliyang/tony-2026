# Kubeflow

## 它是什么

面向 Kubernetes 的机器学习工具箱和 AI reference platform。

## 为什么值得关注

如果你要研究平台化 AI，而不只是单一服务，`Kubeflow` 很值得放进控制塔。

## 核心抽象

- AI reference platform
- training / pipeline / notebook ecosystem
- Kubernetes-native ML toolkit

## 当前建议的学习动作

- 先判断它到底是底座、壳层、平台还是子系统
- 看它和相邻项目是替代关系还是组合关系
- 再决定是本地实验、架构学习，还是生产研究
