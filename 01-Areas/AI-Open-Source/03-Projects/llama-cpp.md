# llama.cpp

## 它是什么

以 C/C++ 为中心的本地 LLM 推理底座，适合理解量化、GGUF 和 edge/local inference。

## 为什么值得关注

如果你想理解“本地模型到底怎么跑起来”，`llama.cpp` 是绕不过去的底座。

## 核心抽象

- GGUF 模型格式
- 量化与 CPU / Metal 本地推理
- 轻量 server / CLI 与多前端生态

## 当前建议的学习动作

- 先判断它到底是底座、壳层、平台还是子系统
- 看它和相邻项目是替代关系还是组合关系
- 再决定是本地实验、架构学习，还是生产研究
