# Linear Algebra for ML

## 为什么重要

- 神经网络几乎处处都在做向量、矩阵和张量运算

## 你要先掌握什么

- 向量：一组数的表示
- 矩阵：线性变换或参数组织方式
- 维度：数据和表示空间的大小
- 矩阵乘法：信息如何被变换和组合

## 学这个是为了什么

- 看懂 embedding、attention、linear layer 等核心组件
