# 云原生安全

这一页后续会重点回答：当系统变成容器化、API 驱动、持续交付和多租户平台之后，安全边界为什么也必须跟着改变。

## 先记住

- 云原生安全不是单独加一个扫描器
- 它会贯穿镜像、供应链、集群、身份、网络、策略和运行时
