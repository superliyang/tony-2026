# Project Intake Checklist

For each project, answer:

- What problem does it solve?
- Why does it matter now?
- Which stack layer does it belong to?
- Is it a `底座`, `壳层`, `平台`, or `子系统`?
- What adjacent projects should it be compared with?
- Is it suitable for local experiments, production study, or both?
- What is the next experiment or reading move?
