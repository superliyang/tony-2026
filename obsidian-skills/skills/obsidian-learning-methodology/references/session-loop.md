# Session Loop

## At The Start

- read the domain README
- read the topic index
- read `Learning Progress.md`
- read `Resume Note.md`

## During The Session

- make changes in the correct layer
- update the nearest index or map when topology changes
- avoid creating isolated notes

## At The End

- refresh progress and resume if the path changed
- note the next recommended reading step
- checkpoint the work if it is coherent and meaningful
