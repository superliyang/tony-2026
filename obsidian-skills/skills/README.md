## 推荐搭配方式

### 研究新领域

1. 先用 `obsidian-domain-research-graph`
2. 再配合 `domain-learning-system`
3. 最后用 `knowledge-vault-research-sync` 做持续维护和 Git 同步

### 维护已有知识库

1. 先扩写内容和索引
2. 更新 `Learning Progress.md` 与 `Resume Note.md`
3. 再用 `knowledge-vault-research-sync` 做校验与 checkpoint

## 这套方法最适合什么场景

- 深入研究一个新领域
- 建立实体 / 主题 / 系统 / 地图分层
- 用 Obsidian 持续沉淀知识图谱
- 支持中断后快速恢复
- 需要定期更新并同步到 GitHub
